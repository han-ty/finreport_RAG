## 目录说明

### 根目录文件
- **README.md**: 项目说明文档
- **config.json**: 项目配置文件（大模型 API 配置等）
- **requirements.txt**: Python 依赖包清单
- **问题清单.txt**: 项目开发过程中遇到的问题记录
- **项目学习路径.txt**: 项目相关的学习资源和路径
- **项目目录结构.md**: 项目目录结构说明文档（本文件）

### backend/
后端代码目录，包含项目的核心业务逻辑。

- **api/**: API 接口模块
  - `server.py`: Web 服务器入口（提供 RESTful API）
  - `__init__.py`: 包初始化文件
- **core/**: 核心功能模块
  - `agent.py`: Agent 智能体，协调各模块工作
  - `config.py`: 配置管理，读取 config.json
  - `database.py`: 数据库连接和查询操作
  - `embedding.py`: 文本向量化处理
  - `knowledge_base.py`: 知识库管理（RAG）
  - `llm_client.py`: 大模型 API 调用封装
  - `pdf_parser.py`: PDF 文档解析
  - `sql_generator.py`: 自然语言转 SQL
  - `visualizer.py`: 数据可视化图表生成
  - `__init__.py`: 包初始化文件

### data/
运行时数据目录，存放生成的数据库文件。
- `financial.db`: SQLite 数据库（存储结构化财务数据）

### frontend/
前端代码目录，基于 React + TypeScript + Vite 构建。
- **src/pages/**: 页面组件
  - `ChatPage.tsx`: 智能对话页面
  - `DatabasePage.tsx`: 数据库查看页面
  - `KnowledgePage.tsx`: 知识库管理页面
  - `SettingsPage.tsx`: 系统设置页面
- **src/services/**: 服务层
  - `api.ts`: API 请求封装
  - `store.ts`: 状态管理


### logs/
日志文件目录，存放各任务运行的日志。
- `task1.log`, `task2.log`, `task3.log`: 各任务运行日志
- `validation_report.txt`: 验证报告

### models/
AI 模型文件目录，存放本地向量化模型。
- `bge-small-zh-v1.5/`: BGE 中文向量化模型

### results/
运行结果目录，存放任务执行的输出结果。
- `B*.jpg`: 可视化图表结果
- `result_*.json`: JSON 格式结果数据
- `result_*.xlsx`: Excel 格式结果数据

### task1/, task2/, task3/
各任务的代码目录，包含独立的任务执行脚本。

### 示例数据/
比赛官方提供的示例数据，包含：
- **附件1**: 中药上市公司基本信息
- **附件2**: 财务报告（上交所、深交所）
- **附件3**: 数据库表名及字段说明
- **附件4**: 问题汇总
- **附件5**: 研报数据（个股研报、行业研报）

## 统计信息

- **主要目录**: 14 个（backend, frontend, data, models, logs, results, task1-3, image, 示例数据, 数据）
- **Python 文件**: 约 16 个（backend 核心模块 + 任务脚本）
- **前端文件**: 约 10 个（TypeScript/TSX 组件）
- **PDF 文件**: 约 37 个（财务报告和研报）
- **Excel 文件**: 6 个（示例数据中的附件）
- **配置文件**: 4 个（config.json, requirements.txt, package.json, tsconfig.json）
